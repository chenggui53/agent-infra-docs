## 🔍 问题分析

### 主要问题
1. **内存不足**: 16GB 内存已使用 98.3%，系统严重依赖压缩内存
2. **CPU 瓶颈**: opencode 和 Terminal 进程占用大量 CPU
3. **磁盘交换**: 大量的内存交换导致磁盘 IO 过高

### 可能原因
- **opencode 工具**: 可能在进行大量的代码分析或编译
- **终端任务**: Terminal 进程可能在运行高资源消耗的任务
- **Chrome 标签页**: 多个 Chrome 标签页占用大量内存
- **系统缓存**: 可能存在内存泄漏或缓存过多的问题
