# 视频自动刷新学习助手

一个浏览器插件，能够自动刷新网页上的视频，帮助学习视频内容。

## 功能特点

- 🌍 **自动刷新**：根据设置的时间间隔自动刷新网页上的视频
- 🎯 **智能识别**：自动识别页面上的视频元素
- 📱 **响应式设计**：支持各种视频网站和布局
- ⚙️ **灵活配置**：可自定义刷新间隔和视频选择器
- 📊 **状态监控**：实时显示插件运行状态
- 🔄 **手动控制**：支持手动触发刷新

## 安装方法

### 开发模式安装（Chrome/Edge）

1. 下载或克隆这个插件仓库
2. 打开浏览器扩展管理页面：
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
3. 开启"开发者模式"开关
4. 点击"加载已解压的扩展程序"
5. 选择插件所在的文件夹
6. 插件安装成功！

### 生产模式安装

当插件开发完成后，可以打包成 `.crx` 文件进行分发。

## 使用方法

1. 点击浏览器工具栏中的插件图标
2. 在弹出的设置页面中：
   - 设置刷新间隔（默认60秒）
   - 配置视频选择器（默认：`video, .video, .video-container`）
   - 启用自动刷新功能
3. 访问包含视频的学习页面
4. 插件会自动识别并刷新视频

## 配置说明

### 刷新间隔

设置视频刷新的时间间隔，单位为秒。范围：10-300秒（默认60秒）。

### 视频选择器

配置插件识别视频的CSS选择器。默认配置包括：
- `video` - 标准视频元素
- `.video` - 常见的视频容器类
- `.video-container` - 常见的视频容器类

可以根据需要添加或修改选择器，多个选择器用逗号分隔。

## 故障排除

### 常见问题

1. **插件无法识别视频**
   - 检查视频选择器配置是否正确
   - 确保页面上有视频元素

2. **刷新不工作**
   - 检查自动刷新开关是否开启
   - 检查刷新间隔设置
   - 查看浏览器控制台是否有错误信息

3. **视频无法播放**
   - 某些网站可能有防自动播放机制
   - 尝试点击视频或页面以触发播放

### 调试方法

1. 打开浏览器开发者工具（F12）
2. 切换到"控制台"标签
3. 查看插件输出的日志信息
